# BLINKIT-Dashboard

📌 Overview:

This project showcases an interactive Blinkit Sales Dashboard developed using Power BI. It focuses on transforming raw grocery retail data into meaningful, actionable insights. The dashboard enables efficient analysis of sales performance, customer preferences, and outlet-level trends through visually appealing and interactive reports.

🎯 Objectives:

✔ Analyze overall sales and performance metrics
✔ Compare outlet types, sizes, and locations
✔ Identify top-performing product categories
✔ Evaluate customer ratings and preferences
✔ Enable dynamic and interactive data exploration

🚀 Key Features:

✨ KPI Cards for quick insights (Total Sales, Avg Sales, Items, Ratings)
📊 Category-wise and fat content analysis
📈 Sales trends and outlet establishment growth
🏪 Outlet size, type, and location comparison
🎛️ Interactive filters and slicers
🎨 Clean, modern, and user-friendly UI

📊 Data Source:

The dataset used in this project is Blinkit grocery sales data stored in Microsoft Excel format.
It includes details such as product categories, outlet types, sales figures, and customer ratings.
The data was cleaned and transformed using Power Query and enhanced with DAX for analysis in Power BI.

📊 Business Impact & Insights:

-> Identifies top-performing products and categories, helping in better inventory planning
-> Analyzes outlet performance based on size, type, and location for strategic decisions
-> Reveals sales trends and patterns to support forecasting and demand planning

-> Improves operational efficiency by highlighting high and low-performing areas
-> Supports data-driven decision-making with clear and interactive visualizations
-> Assists in optimizing pricing and marketing strategies
-> Identifies growth opportunities and areas needing improvement
